### Для запуска
- Linux (тестировал на Mint 20.0 x64);
- Билд приложения и запуск приложения:
```sh
make build
./task04
```
- Билд и запуск приложения одной командой:
```sh
make start
```
- Запуск приложения одной командой:
```sh
make run
```
- Очистка проекта:
```sh
make clean
```
